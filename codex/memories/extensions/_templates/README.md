# Memory extension resource digest templates

These files are templates only. Do not place template filenames inside an extension `resources/` directory.

When creating a real local digest, copy the relevant template to the matching extension's `resources/` directory and rename it with a timestamp prefix:

```text
~/.codex/memories/extensions/harness/resources/2026-05-09T18-30-00-harness-candidates.md
~/.codex/memories/extensions/learnings/resources/2026-05-09T18-30-00-learnings-digest.md
~/.codex/memories/extensions/synesthesia/resources/2026-05-09T18-30-00-synesthesia-signals.md
~/.codex/memories/extensions/negative-ledger/resources/2026-05-09T18-30-00-negative-ledger-digest.md
```

Keep real resource digests local and uncommitted when they include private session details, local paths, repo state, or user-specific evidence.
