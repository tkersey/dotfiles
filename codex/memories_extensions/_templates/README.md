# Memory extension resource digest templates

These are templates only. Do not place these template filenames inside an extension `resources/` directory.

When creating a real local digest, copy the relevant template to the matching extension's `resources/` directory and rename it with a timestamp prefix:

```text
~/.codex/memories_extensions/harness/resources/2026-04-28T18-30-00-harness-candidates.md
~/.codex/memories_extensions/learnings/resources/2026-04-28T18-30-00-learnings-digest.md
~/.codex/memories_extensions/synesthesia/resources/2026-04-28T18-30-00-synesthesia-signals.md
```

Keep real resource digests local and uncommitted when they include private session details, local paths, repo state, or user-specific evidence.
