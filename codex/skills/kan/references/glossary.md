# Glossary

- `K : C -> D`: extension boundary / embedding / schema map.
- `P : B -> C`: projection / observation / handler / public behavior boundary.
- `Lan`: free/generative pushforward.
- `Ran`: coherent observation-based extension.
- `Lft`: left Kan lift / realizing implementation behind `P`.
- `Rft`: right Kan lift / residual obligations behind `P`.
- `Free`: left adjoint/free builder to a projection or forgetful functor.
- `Yoneda`: observation-heavy representation.
- `Coyoneda`: raw payload plus deferred map/path.
- `Defunctionalization`: replacing functions with first-order cases plus interpreter.
- `No exact lift`: required behavior cannot be realized through current `B` and `P`.
