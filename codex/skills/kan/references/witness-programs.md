# Witness programs

Use examples as runnable or inspectable slices, not full implementations.

- `examples/python/finite_kan.py`: finite `Lan`/`Ran` intuition.
- `examples/python/poset_lift_witness.py`: finite-poset lift intuition.
- `examples/python/freyd_lift_boundary_witness.py`: free-builder/template diagnostic.
- `examples/python/defunctionalized_lift_witness.py`: first-order boundary IR.
- `examples/python/yoneda_coyoneda_witness.py`: observation/deferred-generation representation.
- `examples/python/lift_contract_obligations.py`: obligation discovery.
