# `$plan` Source Anchors

The redesign responds to two strong evidence patterns:

1. `$plan` duplicated semantic specification and optimized a large Markdown artifact for lint/convergence ceremony rather than downstream graph compilation.
2. Long execution runs repeatedly lost semantic frontier state across compaction and rediscovered adjacent cases one patch at a time.

The source-boundary model preserves:

```text
$spec-pipeline defines accepted semantics
$plan compiles execution strategy
$st materializes the current commitment horizon
$actuating interprets the policy
```

System-regime handling is informed by the supplied clear/complicated/complex/chaotic systems-thinking transcript.
