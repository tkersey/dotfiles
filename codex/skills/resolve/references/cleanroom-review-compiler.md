# Cleanroom Review Compiler

Review feedback writes to the lab. The compiler writes to delivery.

The delivery branch should not record every review interaction. It should record the smallest patch derived from the counterexample contract.

Core artifacts:

- delivery_freeze / DF-v1
- counterexample_contract / CEC-v1
- review_lab_ledger / RLL-v1
- delivery_patch_recipe / DPR-v1
- ablation_certificate / ABL-CERT-v1
- compiled delivery permit / RGR-CDP-v1
