# Semantic-Surface Conservation

After the review-ready baseline:

```text
silent scope expansion = 0
hard semantic dimensions must not increase
total kernel + realization + proof description must not increase
```

Hard dimensions:

- truth owners;
- public symbols;
- state dimensions;
- protocol cases;
- fallback/compatibility paths;
- wound-specific tests.

A required increase is an explicit complexity decision, not routine review remediation.
