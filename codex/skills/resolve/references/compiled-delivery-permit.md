# Compiled Delivery Permit

`RGR-V4-COMPILED-DELIVERY-PERMIT` authorizes applying the delivery recipe.

It does not authorize ad hoc review-derived mutation.

Required:

- frozen base;
- counterexample contract;
- delivery recipe;
- branch liabilities included;
- non-branch liabilities excluded;
- falsified routes excluded;
- proof matrix;
- ablation required.
