# Resolve Decision Record

Use one default decision record:

```yaml
resolve_decision_record:
  record_version: RDR-v1
  artifact_state:
  review_wave:
  cluster:
  selected_route:
  negative_evidence:
  universalist_check:
  distillation:
  surface_delta:
  proof_matrix:
  implementation_handoff:
  material_improvement:
  gate:
```

Optional sections may be `not-required`, but the record shape remains stable.
