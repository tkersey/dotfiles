# Same-Cluster Stop Rule

After the second same-cluster finding, ordinary point-fix resolution is over.

Allowed routes:

1. `validate-only`
2. `delete-collapse-canonicalize`
3. `normal-form-decision`
4. `review-distillation-mode`
5. `blocked`

`mutate-existing-owner` is allowed only when selected by a normal-form decision that proves it is not another local point fix.

`add-new-surface` is disallowed unless explicit expansion is accepted.
