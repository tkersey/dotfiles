# Boundary Inventory

Required when same owner keeps absorbing semantic branches.

```yaml
boundary_inventory:
  required:
  semantic_surfaces: []
  duplicated_or_parallel_predicates: []
  implicit_table_or_algebra:
  missing_boundary_artifact:
  decision: universalist | reduce | normal-form | distill | blocked
  reason:
```

Use this before `$universalist` escalation. If missing boundary artifact is yes or unknown after recurrence, ordinary owner mutation is blocked.
