# Owner Pressure Gate

Do not ask only whether the owner feels correct.

Measure:

- permits;
- mutations;
- commits;
- patch calls;
- production net;
- validation branches;
- evidence predicates;
- helpers;
- compatibility paths;
- test growth/retirement;
- surfaces retired.

`owner_too_coarse: no` requires measured-below-budget evidence.

A redesign warrant authorizes redesign, not continued ordinary owner growth.
