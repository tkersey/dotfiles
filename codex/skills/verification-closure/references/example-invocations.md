# Example invocations

## Final closure
```md
Use $verification-closure for this task.

Verify the current artifact set using the latest closure handoff packet.
Treat unresolved material soundness as a hard gate.
Return the single highest open gate in Reopen Trigger.
```

## Fast mode
```md
Use $verification-closure in Fast mode for this task.

Return only the contract status, closure gates, readiness, reopen trigger, and closure bottom line.
```
