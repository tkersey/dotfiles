# Productive Cycle Gate

Before a full report, ask whether the watched-session cursor contains a decision-relevant delta.

Full analysis is allowed only for:

- new target-skill evidence;
- new validation failure;
- new user feedback;
- watched session stopped;
- new concrete skill gap;
- new worker evidence that changes interpretation;
- prior finding resolved/falsified;
- explicit user request for full analysis.

Otherwise emit `status-only-stop`.
