# Gate Contract

A pre-spec gate answers whether the objective is decision-complete enough to compile into a spec or hand to planning.

Required handoff sentence:

```text
We are building X, for Y, by changing Z, while explicitly not doing A/B/C, and success means P/Q/R proofs pass.
```

A pre-spec gate never authorizes mutation.

Use `tools/spec_gate.py` for structural evidence; semantic readiness remains a model judgment.
