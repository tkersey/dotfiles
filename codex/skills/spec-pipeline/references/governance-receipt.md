# SGR-v2

`spec_governance_receipt` is the canonical machine-readable artifact for current-spec work.

It contains:

- mode/profile/lane/status;
- authoritative-brief state;
- phase presence;
- gate impact;
- challenge impact;
- fresh-eyes impact;
- subagent accounting;
- plan/mutation authority;
- retro trigger.

A full implementation handoff without SGR-v2 is invalid.
