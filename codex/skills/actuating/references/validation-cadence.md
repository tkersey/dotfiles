# Validation Cadence

## Slice

Run the narrowest proof that directly establishes the selected class/invariant.

Examples:

```text
focused unit/filter
compile-fail fixture
targeted static check
format/check for touched files
```

## Wave

After a coherent aperture/wave:

```text
affected aggregate test
cross-slice invariant proof
build surface affected by the wave
wave-skeptic review when triggered
```

## Ship

On the final current head:

```text
full build
full lint/static
full tests
language-specific closure
acceptance checks
st assert-projection
current proof epoch
```

## Full-suite invalidators

Run full proof earlier when:

```text
repo contract explicitly requires it
build graph/global generated output changed
focused proof cannot close the risk
target/ABI/global state changed
```

## Proof DAG

A proof receipt is reusable only while:

```text
command
artifact/dependency fingerprint
toolchain
target/options
generated artifacts
environment
```

remain valid.

Do not call a prior green command current after an invalidator.
