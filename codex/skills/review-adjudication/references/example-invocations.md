# Example invocations

## Standard
```md
Use $review-adjudication for this task.

Review these PR comments for actual relevance.
Use $seq if the original PR why is missing or disputed.
Return a handoff agenda instead of implementing fixes.
```

## Fast
```md
Use $review-adjudication in Fast mode for this task.

Return only what to act on, rebut, defer, what still needs evidence, and the handoff agenda.
```
