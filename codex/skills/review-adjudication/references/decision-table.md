# Adjudication Decision Table

```text
refuted/stale                         -> reject
unknown                               -> blocked
preference/unrelated                  -> reject
outside horizon                       -> capture follow-up
contract invalidating/scope expanding -> return to contract
novel in-horizon class                -> enter kernel
existing-law nonconformance           -> invalidate realization
additional witness/missing proof      -> attach witness
duplicate                             -> reject or proof-only attach
```

No row directly authorizes mutation.
