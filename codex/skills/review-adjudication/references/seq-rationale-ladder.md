# Seq Rationale Ladder

Use `$seq` when PR rationale is missing, ambiguous, stale, or likely to change adjudication.

Order:

1. current artifacts and review comments
2. `plan-search` / `artifact-search`
3. `find-session` + `session-prompts`
4. `memory-map` / `memory-provenance` / `memory-history`
5. memory summaries as weak context only

Never use memory to override current artifacts.
