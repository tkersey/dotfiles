# Seq rationale ladder

Use `$seq` when the PR why is missing, disputed, stale, or likely to change
adjudication.

Preferred ladder:

1. `plan-search`
2. `artifact-search`
3. `find-session` + `session-prompts`
4. `memory-map`
5. `memory-provenance`
6. `memory-history`

Use `$seq` to recover rationale, not to manufacture obligations. Current
artifacts still decide whether a comment is grounded, stale, in scope, or
actionable.
