# Example invocations

## Full review
```md
Use $adversarial-reviewer for this task.

Perform a full-scope de novo adversarial review of the current artifact set.
Find material issues, make the required changes explicit, and end with a typed Change Agenda.
```

## Fast mode
```md
Use $adversarial-reviewer in Fast mode for this task.

Review the current patch and return only material findings, the soundness ledger, the change agenda, and the reviewer bottom line.
```
