# Tune Gap Taxonomy

```text
activation         trigger present, skill missed
interpretation     skill active, rule misunderstood
workflow           steps/order/handoff fail
tooling            repeated manual workaround or missing CLI
resource           missing reference/script/asset
observability      no reliable outcome signal
metadata           stale trigger/default prompt
boundary           companion ownership confused
source-scope       wrong evidence population
decision-contract  rules cannot be stably observed
observability      decision effect cannot be recovered
outcome            compliant route repeatedly fails/reopens
ceremony           activation with no material delta
overconstraint     skill blocks useful routes without evidence
```
