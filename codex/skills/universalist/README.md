# universalist

Drop-in replacement skill for universal-construction-driven codebase architecture.

It keeps the original Universalist intent—choose the smallest honest construction—and adds a Track D for universal architecture boundaries: free syntax, coherent observations, transported semantics, lifted implementations, explicit IR, and law tests.

## Install

From your repo root:

```bash
rm -rf codex/skills/universalist
unzip universalist-dropin-replacement.zip -d .
cd codex/skills/universalist
chmod +x scripts/*.sh
./scripts/check_universalist.sh
```

## Use

Ask for `$universalist` when code smells indicate a structural refactor rather than an ordinary fix.
