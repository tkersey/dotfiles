# Freyd/AFT Boundary Diagnostic

Use after a lift-shaped boundary appears.

Question:

```text
Is P : B -> C0 disciplined enough to support a canonical Free : C0 -> B?
```

Checklist:

1. `P` is concrete.
2. `B` can combine constraints.
3. `P` preserves those constraints enough to test.
4. There is a bounded family of implementation templates.
5. `Free` has a projection law.

If not, produce an obstruction report rather than pretending a builder exists.
