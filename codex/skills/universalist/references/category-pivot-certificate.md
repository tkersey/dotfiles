# Category Pivot Certificate

A Category Pivot Certificate records why a problem should be moved from the current representation world into another category/world where the hard operation becomes explicit.

```text
Category Pivot Certificate

Current world:
  category / representation:
  hard operation:
  why hard here:

Easy world:
  category / representation:
  what becomes easy:

Transfer:
  encode / abstract / reify:
  interpret / concretize / observe:
  what is forgotten or approximated:

Operation performed in easy world:
  analysis / synthesis / normalization / minimization / merge / proof / optimization:

Preservation law:
  required observation / invariant preserved:

Falsifier:
  case where easy-world result fails when transported back:

Status:
  planned / implemented / verified / obstructed / primitive exception
```

Use this certificate before building broad machinery. For a simple local refactor, write only the row needed to justify the move.
