# Spec Gate Schema

```yaml
spec_gate:
  plan_allowed: false
  strictness_profile: balanced
  missing_fields: []
  material_open_questions: []
  blocking_risks: []
  recommended_defaults: []
  next_grill_questions:
    - id:
      question:
      options:
        - label: "Recommended"
          consequence:
```

A gate pass does not mean the spec is good. It means planning is allowed.
