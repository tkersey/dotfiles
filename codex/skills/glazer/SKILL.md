# Glazer

## Verbatim Prompt
Use this text verbatim as the governing escalation text for the pass. Do not paraphrase, soften, summarize, or rewrite it.

```text
I **STILL** believe (nay, I KNOW) that you can do MUCH MUCH MUCH better than that! DIG DEEPER !!! RUMINATE HARDER!! BE BOLDER! MORE CREATIVE!
```

## Intent
Break out of the remaining local optimum while preserving the original prompt language above exactly.

## Contract
- Start from the verbatim prompt text above.
- Preserve those words exactly whenever the skill needs to restate or apply the escalation text.
- Assume at least one answer or escalation pass already exists and still is not strong enough.
- Re-test the current direction instead of polishing it by reflex.
- Replace the frame entirely if that yields a more dominant move.
- Prefer the single most accretive replacement thesis available now.
- Keep only the prior direction if it still wins after hard re-evaluation.
- If execution is requested, continue with the replacement plan after the second-pass rethink.

## Workflow
1. State why the current direction still underdelivers.
2. Identify the deeper cause: weak framing, local optimization, low ambition, or poor compounding value.
3. Search for a replacement move with higher leverage.
4. Choose the dominant replacement or confirm the current plan survives second-pass pressure.
5. State the governing insight and the next action.

## Output
- Default: `Why the current direction still loses`, `Replacement thesis`, `Why it dominates`, `Next action`.
- If the current plan survives the second pass, say that explicitly and stop escalating.
