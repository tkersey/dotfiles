# Intent-Closed Resolve Handoff

When called by `$resolve`, the driver receives only:

```text
current AC/CEB/MBK/RC fingerprints
selected design
campaign-base-pinned worktree
permitted owners and surfaces
surfaces to retire
retained factor map
quotient relation
preservation/recomposition rules
hard surface ceiling
proof laws
```

It must not receive raw review prose.

If realization discovers:

```text
same-class counterexample
new in-horizon class
outside-horizon requirement
new owner/state/protocol/public/fallback distinction
```

return it as a new observation and stop.

Do not append-patch.

The controller decides whether to invalidate realization, return to kernel, return to contract, or defer.
