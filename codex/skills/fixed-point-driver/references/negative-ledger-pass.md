# Negative Ledger Pass

Record a pass with:

- `phase`: preflight / post-remediation / post-review / pre-closure / capture / handoff
- `mode`: query / map / capture / reopen / handoff / none
- `artifact_state_id`
- `topical_query`
- `sources_checked`
- `result`
- `durable_capture`

A completed pass with no matching evidence records `no-applicable-negative-evidence`; it is not skipped.
