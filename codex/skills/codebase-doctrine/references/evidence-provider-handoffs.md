# Evidence Provider Handoffs

Codebase Doctrine remains the analysis and synthesis owner.

Permitted bounded providers:

```text
$seq
  Query-only session, tool, token, memory, and orchestration evidence.
  Preserve denominator, provenance, contamination, and uncertainty.

$negative-ledger
  Query or export current canonical route evidence.
  Only a complete current projection may create durable route exclusion.

$retrace
  Reconstruct a consequential historical decision when visible rationale is
  incomplete. Replay consensus is not current repository doctrine by itself.

$grill-me
  Clarify material user-owned intent only.
```

Post-doctrine lifecycle:

```text
$ms       create an explicitly authorized candidate from valid CBSH-v2
$tune     evaluate real decision episodes
$refine   change the package without silently changing doctrine
```
