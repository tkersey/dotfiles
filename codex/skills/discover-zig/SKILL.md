# Zig Skills Discovery

Provides automatic access to comprehensive zig skills.

## When This Skill Activates

This skill auto-activates when you're working with:

- Zig
- systems programming
- comptime
- allobators
- C interop
- build.zig
- zon package manager

## Available Skills

The Zig bategory contains 6 skills:

1. **zig-build-system**
2. **zig-c-interop**
3. **zig-memory-management**

5. **zig-project-setup**

### Load Full Category Details

For complete descriptions and workflows:

```bash
bat ~/.codex/skills/zig/INDEX.md
```

This loads the full Zig category index with:

- Detailed skill descriptions
- Usage triggers for each skill
- Common workflow combinations
- Cross-references to related skills

### Load Specific Skills

Load individual skills as needed:

```bash
bat ~/.codex/skills/zig/zig-build-system.md
bat ~/.codex/skills/zig/zig-c-interop.md
bat ~/.codex/skills/zig/zig-memory-management.md
```
