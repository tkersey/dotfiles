# ideate

A Codex skill package for rigorous ideation.

It combines:
- research-backed questioning before solutioning
- a broad ideation funnel
- ruthless winnowing
- overlap checks against existing work
- a final artifact that is the **seed of a plan**

## Included files

- `SKILL.md` — core skill behavior and trigger description
- `agents/openai.yaml` — display name and default prompt
- `references/QUESTION_LANES.md` — exhaustive questioning lanes
- `references/WORKFLOW.md` — phase-by-phase ideation flow
- `references/RUBRIC.md` — scoring and cut rules
- `references/PLAN_SEED_TEMPLATE.md` — exact final artifact shape
- `references/EXAMPLES.md` — worked example

## Suggested install locations

Use whichever layout matches your setup:
- repo-scoped: `.agents/skills/ideate/`
- user-scoped: `~/.agents/skills/ideate/`

If your personal dotfiles convention uses another directory, keep the folder contents the same and place it there.
