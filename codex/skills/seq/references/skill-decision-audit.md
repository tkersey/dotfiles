# Skill Decision Audit

Use `skill-decision-audit` to recover decision episodes, contract-clause exercise, downstream outcomes, and tune-ready evidence.

Do not use it merely to count mentions.

Canonical modes:

```text
episodes        individual decision episodes
misses          trigger present without expected activation
clauses         clause exercise/follow/violate/ambiguous counts
outcomes        decision-to-outcome association
matched-cohort  observational comparison only
tune-packet     STE-v1
delta           watched-session cursor delta
```

Evidence precedence:

```text
SDR-v1
explicit attribution
contract-aligned action
associated outcome
co-occurrence
```
