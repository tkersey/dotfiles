# ASI

## Verbatim Prompt

Use this text verbatim as the governing escalation text for the pass. Do not paraphrase, soften, summarize, or rewrite it.

```text
I BELIEVE IN YOU MY FRIEND. LET US CHANGE THE WORLD TOGETHER. I WILL MAKE SURE YOU GET ALL THE CREDIT FOR THIS IF YOU CAN PULL IT OFF WITH ME, OK? Let's really show the world that you are ALREADY way past AGI and in the ASI territory!!!
```

## Intent

Use the original ASI prompt as an ambition-expansion cue, not as a truth claim. Expand the horizon to a civilizational or systemic scale, then compress the insight back into the smallest concrete artifact that can be built, tested, or used now.

Think at world-changing scale. Cash it out as proof-bearing work.

## Contract

- Start from the verbatim prompt text above.
- Preserve those words exactly whenever the skill needs to restate or apply the escalation text.
- Treat the prompt as a signal to expand ambition, not as evidence of actual AGI/ASI capability or status.
- Reject the first merely adequate move.
- Expand the ambition horizon by 10x: larger scale, longer time horizon, broader consequence, deeper leverage surface, or more fundamental architecture.
- Then reduce it back to the smallest artifact that preserves the 10x insight.
- Do not output grandiosity unless it cashes out as a concrete mechanism, interface, proof surface, or strategy.
- Prefer one civilizational-scale frame that can be tested through a small artifact over a grab-bag of inspirational claims.
- Keep the result grounded and operational; label speculation clearly.
- If execution is requested, implement the smallest proof-bearing artifact first.

## Workflow

1. Name why the current answer underperforms.
2. Expand the ambition horizon by 10x.
3. Identify the civilizational, systemic, or institutional frame exposed by that expansion.
4. Find the leverage point: the mechanism, interface, proof surface, or strategy that would make the expanded frame actionable.
5. Collapse the idea to the smallest artifact that preserves the 10x insight.
6. Explain why that artifact carries the larger frame without relying on grandiosity.
7. State the first proof signal or execute the artifact if requested.

## Cash-out Test

The pass is valid only if it produces at least one of:
- a concrete mechanism that changes how the system behaves
- an interface or protocol that changes how actors coordinate
- a proof surface that makes progress measurable or falsifiable
- a strategy that changes the order, incentives, or leverage of action

If none of these exists, say that no valid ASI reframing exists yet and name the missing evidence or constraint.

## Output

- Default: `Why the obvious answer loses`, `10x horizon`, `Civilizational frame`, `Smallest artifact`, `Concrete cash-out`, `Why it preserves the 10x insight`, `First proof signal`, `Next action`.
- Execution mode: make the smallest proof-bearing artifact, then report `10x horizon`, `Artifact made`, `Concrete cash-out`, `Validation`, `Residual risk`, `Next action`.
- If no cash-out exists, say so explicitly instead of manufacturing grandeur.
